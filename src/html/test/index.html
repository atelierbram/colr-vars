Usage

In the main sass stylesheet import the sass-maps color variables colrvars.scss

@import 'colrvars';

.test {
  background-color: color(blue-11);
}
.test p {
  color: color(gold-06);
}

Alternatively, convert the sass-map syntax to the dollar-sign syntax like:

@import 'colrvars';

$blue-11: color(blue-11);
$gold-06: color(gold-06);

.test {
  // background-color: color(blue-11);
  background-color: $blue-11;
}

.test p {
  // color: color(gold-06);
  color: $gold-06;
}
